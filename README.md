# Fraud Detection System (ML + Flask)

## 📌 Project Overview
This project is an end-to-end fraud detection system built using:
- Machine Learning (XGBoost)
- Flask API
- SQLite database
- Modular preprocessing pipeline

The system predicts whether a transaction is fraudulent and logs all predictions for future retraining and analytics.

---

## ⚙️ Tech Stack
- Python
- Scikit-learn
- XGBoost
- Flask
- SQLite
- Pandas

---

## 🧠 ML Workflow
1. EDA & preprocessing
2. Feature engineering & encoding
3. Model training (Logistic, RF, XGBoost)
4. Model comparison & selection
5. Deployment via Flask API
6. Prediction logging for retraining
